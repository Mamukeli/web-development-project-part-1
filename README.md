# Boost Force Mobile Car Wash Website

## Project Overview

Boost Force Mobile Car Wash is a responsive multi-page website developed for a mobile car wash business based in Cape Town, South Africa. The website was created as part of the WEDE5020 Web Development Introduction assignment project.

The platform allows users to:

* View car wash services
* Learn about the business
* Contact the company
* Book mobile car wash appointments online

The website focuses on modern responsive design, professional branding, and improved user experience.
