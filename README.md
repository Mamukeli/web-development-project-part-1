## 📌 Project Overview
Boost Drive Mobile Car Wash is a mobile cleaning service that provides convenient car washing at customers' locations such as homes, offices, and residential complexes. The website is designed to make booking services quick and easy.

---

## 🎯 Target Audience
- Office workers
- Homeowners
- Residents in complexes
- Busy professionals

---

## 🌐 Website Pages

### 1. Home Page (index.html)
- Introduction to the business
- Overview of services
- Call-to-action buttons

---

### 2. Services Page (services.html)
This page includes:
- Basic Wash
- Full Exterior Wash
- Interior Cleaning
- Full Premium Package

Each service includes a short description and pricing.

---

### 3. Booking Page (booking.html)
This page allows customers to:
- Enter name and contact details
- Select service type
- Choose preferred date and time
- Submit booking request

---

### 4. Contact Page (contact.html)
- Business contact details
- Location info
- Contact form

---

## 🎨 Design Features
- Mobile-friendly layout
- Clean and modern design
- Bold colour scheme (black, yellow, white)
- Easy navigation menu
